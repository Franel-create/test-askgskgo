# Тестовое задание для ArticMedia.
Онлайн демо: https://crimson-goldfish.static.domains/

## Инструкция по работе с проектом

### Установите пакеты
```
npm install
```

### Запуск в режиме разработки
Для запуска проекта в режиме разработки используйте команду:

```
npm run dev
```

### Сборка проекта

```
npm run build
```
